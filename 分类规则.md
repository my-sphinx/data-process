＃分类规则 
1. **只涉及用户自选股列表、自选股票相关问题 -> `自选`**
   示例：`我的自选股今天表现如何？`、`打开我的自选股`
   错误示例：`交易页面新股新债的相关问题`
2. **查询用户账户资产、总资产、可用资金等，不包含其他资产相关问题的询问 -> `资产查询`**
   示例：`我的总资产多少？`、`帐户余额`
   错误示例：`资产盈亏功能的作用`    
3.**必须含有明确买入、卖出、撤单、委托等语义的操作动词，不包含股票交易相关问题的询问 -> `交易`**  
   示例：`买入贵州茅台`、`帮我卖出1909股宁德时代`
   错误示例：`股票连续跌停怎么快速转出？`、`对手方最优市价订单申报无对手方未成交订单`
4.**查询用户当前的持仓股票信息，不包含其他股票持仓相关问题的询问 -> `查持仓`**
   示例：`我的持仓有哪些？`、`查看持仓`
   错误示例：`交易页面持仓或可用的相关问题`、`持仓市值计算公式`
5. **当股价达到某数值时提醒 -> `到价提醒`**
   示例：`$比亚迪＄跌到 266 时提醒我`
6. **明确对某只股票近期所有的资金流入、流出进行分析的相关提问 -> `股票资金流向分析`**
   示例：`$五粮液$ 近5日主力资金流向`
7.**只通过资金流入/出情况选出某一只或多只股票的选股问题 -> `资金流向选股`**
   示例：`连续3天主力净流入的股票`  
8.**询问开户营业部、所有昔业部的地址、联系电话、席位号等相关信息 -> `查开户背业部`**
   示例：`我在哪个营业部开户的？`
9.**对股票、基金、指数/行情进行诊断、询问、评价、宏观分析，以及选股相关问题 -> `问诊股`**
   示例：`芯片etf近期表现`、`中国货币政策调整会对股市产生什么影响`
10.**浴询业务办理、证件更新、活动/交易规则，科普类问题等非交易问题 -> `咨询客服`**
   示例：`身份证如何更新？`、`股票如何查看诊股信息`、`什么是国债逆回购？`、`如何利用换手率选择牛股`
11.**与上述类别无关的非金融意图、广告、活动介绍、闲聊等 -> `拒识`**
   示例：`春节红包活动规则`、`世界杯竞猜`
